# React Rich Painter

**React Rich Painter**は、Reactアプリケーションに本格的なペイント機能を統合できるライブラリです。

## ✨ 主な機能

- 🖌️ **ブラシ機能** - 筆圧対応、カスタムブラシ、透明度調整
- 🎨 **カラーパレット** - 直感的な色選択
- 📚 **レイヤー機能** - 複数レイヤーの管理、表示/非表示、並び替え
- 🔧 **ツール** - ペン、消しゴム、投げ縄選択、スポイト、手のひらツール
- 📱 **マルチ入力対応** - マウス、タッチ、ペンタブレット
- 💾 **Import/Export** - 描画状態のJSON保存・復元

## 📦 対応環境

- React 19.x
- Node.js 22以上

## 🎯 ユースケース

- **イラスト作成アプリ** - フル機能のPaintingプリセットを使用
- **メモ・スケッチアプリ** - シンプルなNotebookプリセットを使用
- **画像アノテーション** - レイヤー機能を活用
- **教育用ツール** - タッチ対応で直感的な操作
